## 页面定位

这页应该是“数据层”的解释页，不是“结论页”。

它要让观众看到 Method 03 的实地实验不是只做主观问卷，而是会形成一个多模态数据包：

- 视觉注意：眼动注视点与注视时长
- 空间行为：参与者路线轨迹与停留点
- 身体反应：姿态、方向、停顿、回看
- 主观负荷：认知负荷评分
- 记忆结果：遗忘曲线与延迟回忆数据

核心信息：

**Behavioral traces + subjective load + delayed memory together validate the design principles.**

日文辅助句：

**行動軌跡・主観的負荷・遅延記憶を組み合わせ、設計原則の効果を検証する。**

## 建议视觉结构

建议沿用当前方法 3 的暗色系统，保持和第 13、14 页连续：

- 背景：dark
- 主色：black / white / lemon green
- 页面风格：瑞士图形现实主义、扁平、数据仪表感
- 避免：装饰性卡片顶部线、渐变背景、圆角卡片、过度说明文字

推荐布局：

### 左侧：实验场景与采集逻辑

占宽约 `38%`。

内容包括：

- 大标题：`DATA CAPTURE`
- 副标题：`Controlled field experiment`
- 一个简化的路径图或场地线图
- 路径上叠加若干观测点：
  - gaze
  - route
  - posture
  - load
  - recall

视觉上可以用黑底白线，柠檬绿点作为采集节点。

### 右侧：五类数据模块

占宽约 `62%`。

使用五个横向数据条或 2+3 网格模块。推荐 5 行横向模块，因为它更像数据采集清单，阅读顺序更清楚。

每行结构：

- 左侧 icon
- 数据类型英文标题
- 一句可测指标
- 日文短句
- 右侧小型数据可视化 mini glyph

不要做传统大卡片堆叠。模块之间用负空间、编号、细线或色块区分。

## 数据模块内容

### 01 Eye-tracking fixation data

英文：

**Eye-tracking fixation data**

指标：

`fixation point / dwell time / scan path`

日文：

**注視点・注視時間・視線経路**

建议图形：

小型植物牌轮廓上叠加 5-7 个柠檬绿注视点，点大小代表停留时间。

用途说明：

用于判断不同设计原则是否改变注意分配、是否减少无效搜索。

### 02 Visitor route trace data

英文：

**Visitor route trace data**

指标：

`walking path / stop point / revisit`

日文：

**移動経路・停止点・再訪行動**

建议图形：

一条折线路径穿过 3 个 signage 节点，节点用柠檬绿高亮。

用途说明：

用于判断信息布局是否影响路径选择、停留顺序与探索效率。

### 03 Body posture and orientation data

英文：

**Body posture and orientation data**

指标：

`viewing angle / standing distance / body turn`

日文：

**視認角度・距離・身体方向**

建议图形：

极简人体轮廓 + 面向标牌的角度扇形。

用途说明：

用于捕捉参与者是否需要调整身体姿态才能理解标牌，间接反映可读性与交互负荷。

### 04 Cognitive load rating

英文：

**Cognitive load rating**

指标：

`NASA-TLX / perceived difficulty / mental effort`

日文：

**認知負荷・難易度・精神的努力**

建议图形：

水平刻度条，左低右高，柠檬绿条表示评分。

用途说明：

用于连接设计变量与主观负荷，验证“降低认知负荷”的目标。

### 05 Forgetting curve and memory retention

英文：

**Forgetting curve and memory retention**

指标：

`immediate recall / next-day recall / delayed recall`

日文：

**直後再生・翌日再生・遅延再生**

建议图形：

三点折线或递减曲线，重点标注 retention drop。

用途说明：

用于验证设计原则是否真正改善记忆保持，而不是只改善即时理解。
