THE THEORETICAL GAP

Three theories.
No system model.

三つの理論。しかし、システム水準のモデルは存在しない。
01

Contextual Learning

文脈的学習モデル

Learning is shaped by place and experience.

02

Attention–Value

注意・価値モデル

Visitors attend when value feels worth the effort.

03

Cognitive Load

認知負荷理論

Working memory has limited capacity.

04

Missing System Level

システム水準の欠落

Effects across a sequence of signs remain unknown.

05

Missing Memory Link

記憶への接続不足

Cumulative load is rarely linked to retention.

06

Missing Translation

設計原則への変換不足

No clear path from model to agreed design rules.
