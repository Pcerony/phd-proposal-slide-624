# 研究目标部分 Slide 规划指南
> 对应演讲稿 Section 2「Research Objectives」，共计 1 张 Slide。
> 核心定位：纯粹展示「我要拿到什么成果」，绝不触碰「怎么做」，为下一张方法论 Slide 留下悬念。

---

## Slide — 研究目标（Research Objectives）

### 整体布局方案
三成果垂直排列（或扇形展开），每个成果为一个独立卡片，
由上至下依次动画进入，节奏感强，便于评委逐条跟进演讲节奏。

---

### 顶部标题区

```
EN:  What This Research Aims to Produce
JP:  本研究が産み出すもの
```

---

### 成果卡片 ①

**图标建议：** 网络/节点连线图（暗示"因果关系"）

```
EN:  A  CAUSAL MODEL
     ——————————————————————————————————————
     Revealing how spatial information
     design variables connect to
     visitor cognition and memory.

JP:  因果メカニズムモデル
     ——————————————————————————————————————
     空間情報デザインの変数が
     来訪者の認知・記憶と
     どうつながるかを明らかにする。
```

---

### 成果卡片 ②

**图标建议：** 规则/检查清单图标（暗示"可直接使用的原则"）

```
EN:  MEMORY-ORIENTED  DESIGN PRINCIPLES
     ——————————————————————————————————————
     Not just theory —
     principles that designers
     can directly apply.

JP:  記憶指向デザイン原則
     ——————————————————————————————————————
     理論にとどまらず、
     デザイナーが実務で
     直接活用できる原則。
```

---

### 成果卡片 ③

**图标建议：** 验证印章 / 实地勘测图标（暗示"真实环境检验"）

```
EN:  A  VALIDATION FRAMEWORK
     ——————————————————————————————————————
     Ensuring the principles are tested
     in a real environment —
     not just a lab assumption.

JP:  測定・検証フレームワーク
     ——————————————————————————————————————
     デザイン原則が
     実際の環境で検証されたものであることを
     担保する体系。
```

---

### 底部衔接语（小字，过渡用）

```
EN:  How exactly I will get there, step by step —
     that is what the next slide will show.

JP:  どのようにしてこの三つを手に入れるか——
     その具体的なプロセスは次のスライドで。
```

---

### 动画节奏设计

| 时序 | 动画内容 |
|------|---------|
| 0.0s | 顶部标题淡入 |
| 0.5s | 成果卡片①从左侧滑入，稳定显示 |
| 演讲者说完①后按键 | 成果卡片②从左侧滑入 |
| 演讲者说完②后按键 | 成果卡片③从左侧滑入 |
| 成果③稳定后 | 底部衔接语淡入（字体较小、颜色偏灰） |

> 每张卡片「按键触发」而非自动播放，让演讲者完全掌控节奏。

---

### 视觉设计建议

| 元素 | 建议 |
|------|------|
| **背景** | 深色系，与封面保持一致的设计语言 |
| **卡片样式** | 圆角矩形，带左侧彩色边框线（三张用不同颜色区分：蓝 / 绿 / 橙） |
| **成果关键词** | CAUSAL MODEL / DESIGN PRINCIPLES / VALIDATION FRAMEWORK 使用超大字体、品牌色高亮 |
| **英文为主** | 字号大；日文为辅，置于英文正下方，字号约为英文的 75% |
| **图标** | 简洁线性风格，与文字左对齐，避免过度装饰 |
| **底部衔接语** | 浅灰色小字，居中或靠右，视觉上「低调」但语气上「勾人」 |

---

### 与前后 Slide 的连接逻辑

```
← Slide 上一张（研究背景·核心问题）
   提出痛点：48% 遗忘率 + 核心问题
   ↓
★ 本张 Slide（研究目标）
   回答：我要产出什么来解决这个问题
   ↓
→ Slide 下一张（研究方法·3W）
   回答：我具体怎么一步步得到它
```
