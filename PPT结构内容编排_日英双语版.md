## Slide 03 — 研究背景②：既存研究の盲点（Background · The Gap）
**対応スクリプト：** 研究背景【約25秒、第2段落前半】

### タイトル
```
EN: The Gap: Research Stops at ONE Sign
JP: 盲点：既存研究は「一枚のサイン」しか見ていない
```

### 対比表（双カラム）
| 既存研究が扱うもの | 現実の場面で欠けているもの |
|---|---|
| EN: Single sign design | EN: The full path of signs |
| JP: 単一サインのデザイン | JP: 来訪者が実際に歩くサインの連続した経路 |
| EN: Immediate attention | EN: Cumulative cognitive load |
| JP: 即時の注意 | JP: 蓄積された認知負荷 |
| EN: Isolated cognition | EN: Sequential memory degradation |
| JP: 単独の認知 | JP: 連続的な記憶の低下 |

### キーワード補足
```
EN: "Museum Fatigue" — cognitive overload across a series of signs
JP: 「博物館疲労（Museum Fatigue）」— サインを連続して見ることによる認知過負荷
```

### ビジュアル設計提案
- 来訪者が Sign 1 → 2 → 3 → 4 → 5 と順番に歩くシンプルな図解
- 矢印や数字が5に近づくにつれ色が赤く・重くなる演出
- 「Museum Fatigue」をキーワードとして枠線付きで強調表示

## Slide 04 — 研究の核心問題（Core Research Question）
**対応スクリプト：** 背景結末【約25秒】

### 衝撃データ（最大インパクト）
```
48%
```

### 英語説明（メイン）
```
Average second-day forgetting rate
among botanical garden visitors

— Master's Research, Forgetting Curve Test
```

### 日本語補足（サブ）
```
修士研究での遺忘曲線テストにより判明：
植物園来訪者が解説サインの情報を翌日忘れる割合は平均48%

認知負荷以上に深刻な課題 → 情報留存率（Information Retention）
```

### 核心問題（中央、大きく）
```
EN: How to design an interpretive signage system
    that reduces cognitive load
    and improves information retention?

JP: 認知負荷を下げ、情報留存率を高める
    解説サインシステムを、
    どう設計すればよいのか？
```

### ビジュアル設計提案
- 「48%」を画面いっぱいの超大フォント＋アクセントカラー（オレンジレッドや警告イエロー）
- 核心問題は引用符または色付きボックスで囲む
- 版面の余白を広くとり、「問いの重さ」を演出

## Slide 05 — 研究目標（Research Objectives）
**対応スクリプト：** 研究目標【約50秒】

### タイトル
```
EN: Three Research Objectives
JP: 三つの研究目標
```

### 目標①
```
EN: ① REVEAL
    How system-level factors
    (information density, narrative order, spatial proximity)
    causally affect visitor memory retention.

JP: ① 解明する
    情報密度・物語的経路・空間的近接性などの
    システムレベルの変数が、来訪者の情報保持度に
    どのような因果メカニズムで影響を与えるかを明らかにする。
```

### 目標②
```
EN: ② CO-CREATE
    Turn the cognitive model into memory-oriented
    design principles through multi-stakeholder
    co-creation workshops.

JP: ② 共創する
    来訪者・園内管理者・科学コミュニケーター・
    デザイナーとの共創ワークショップを通じて、
    認知因果モデルを記憶指向のデザイン原則へと転換する。
```

### 目標③
```
EN: ③ VALIDATE
    Develop tools & experiments to test the design
    principles in the Fukuoka City Botanical Garden greenhouse.

JP: ③ 検証する
    福岡市植物園温室において、開発した測定ツールと
    実験計画によりデザイン原則の実効性を検証する。
```

### ビジュアル設計提案
- 三目標を縦並びまたは横並びのカード形式で表示
- キーワード（REVEAL / CO-CREATE / VALIDATE）を大文字・太字・ブランドカラーで強調
- 日本語はそれぞれの英語の直下に小さめで並べて表示

## Slide 06 — 研究方法（Methodology）
**対応スクリプト：** 研究方法【約85秒】

### タイトル
```
EN: Three-Phase Sequential Research Design
JP: 三段階シーケンシャル研究デザイン
```

### Phase 1（左カラム）
```
EN: Phase 1  ·  Year 1, Semester 1
    Exploratory Causal Modeling
    • Literature review
    • Stakeholder interviews (visitors, managers…)
    • Field walkthrough at Fukuoka greenhouse
    → Output: Initial Causal Model

JP: 第1フェーズ  ·  1年次前期
    探索的認知因果モデリング
    • 文献レビュー（環境心理学・認知負荷理論）
    • 多方利害関係者へのインタビュー
    • 福岡温室でのフィールドウォークスルー
    → 成果：初期因果モデル
```

### Phase 2（中央カラム）
```
EN: Phase 2  ·  Year 1, Semester 2
    Co-Creative Translation
    • 2 co-creation workshops (8–10 people each)
    • Scenario storyboarding
    • Participatory prototyping
    → Output: First Design Principle hypotheses

JP: 第2フェーズ  ·  1年次後期
    共創的転換
    • 共創ワークショップ×2回（各8〜10名）
    • シナリオストーリーボーディング
    • 参加型プロトタイピング
    → 成果：デザイン原則仮説（第一弾）
```

### Phase 3（右カラム）
```
EN: Phase 3  ·  Year 2–3+
    Empirical Validation
    • Single-sign → Global multi-sign sequential
      eye-tracking system
    • ~60 participants in real greenhouse
    • NASA-TLX: cognitive load data
    • Forgetting curve tests: retention data
    → Output: Validated Design Principles

JP: 第3フェーズ  ·  2〜3年次以降
    実証的検証
    • 単一サイン用アイトラッキング →
      全体・多サイン連続アイトラッキングシステムへ進化
    • 約60名の被験者が実際の温室内で
      一連の解説サインを巡覧・閲読
    • NASA-TLXスケール：主観的認知負荷データ収集
    • 遺忘曲線テスト：情報留存データ収集
    → 成果：検証済みデザイン原則
```

### 底部主線（スパンするアロー）
```
EN: Cognitive Modeling  →  Design Principle Translation  →  Design Principle Validation
JP: 認知メカニズムモデリング  →  デザイン原則の転換  →  デザイン原則の検証
```

### ビジュアル設計提案
- 横並び三列カード形式
- 各カラム上部の色帯で段階を区別（青→緑→オレンジ）
- 底部に三列を貫く主線矢印（グラデーション）

## 附：言語レイアウト原則

```
┌──────────────────────────────────────────┐
│  English Main Text (Large, Primary)      │
│  メインの英語テキスト（大・主）             │
│                                          │
│  日本語補足テキスト（小・副）               │
```

- キーワード・術語は「English（日本語）」の形式で並記
